# 皮肤设计实验室

这里保存 Workmode 自制、可追踪的静态视觉样稿，用于比较材质、排版、几何、图标和结构外壳。样稿不连接后端，也不代表已经内置或可由当前皮肤协议完整表达。

目录约定：

- 每个方案使用独立目录，入口统一为 `index.html`；
- 可合法再分发的共享字体放在 `_assets/`，并随附许可证；
- 外部皮肤解包、游戏截图、论文和版权状态不清楚的参考素材不得进入本目录；
- 方案进入产品前，应拆成 `workmode-skin/v3` 可表达的积木，并补协议、运行时、测试和文档；
- 已成为正式无资源示例的清单放在 `examples/skins/`，不要把静态 HTML 当作可导入皮肤包。

当前样稿包括紫晶星象塔、奶油泡芙、霜核、夜场调音台、水墨书斋、午夜事务所、绿磷终端和研天雪主题探索。它们用于设计回归和后续积木提炼，可以直接用浏览器打开各目录的 `index.html`。

当前 v3 的逐项覆盖和下一批通用积木优先级记录在 [`docs/SKIN-LAB-MATRIX.md`](../../docs/SKIN-LAB-MATRIX.md)。
