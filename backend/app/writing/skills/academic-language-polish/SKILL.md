# Skill: academic-language-polish

只处理用户给出的原文。改善语法、用词、简洁度、学术语气和句间衔接，但不得新增事实、数据、引用、方法、结论或作者立场。

执行时遵守：

- 专有名词、术语、样本量、统计值和引用编号必须保持不变；明显存在疑问时保留原文。
- 不机械消除所有被动语态；是否改写取决于表达是否更清楚，以及是否会改变研究责任主体。
- 不把相关关系增强为因果关系，不把可能性增强为确定性，不把局部结果扩大成普适结论。
- 化学式、数学指数、单位与变量下标优先输出为可直接粘贴到 Word 的 Unicode 字符，例如 H₂O、CO₂、10⁻³ mol·L⁻¹、R²、xᵢ。
- 不输出 HTML、LaTeX、`SUP/SUB` 标记、修改说明或代码块；只输出润色后的正文。
