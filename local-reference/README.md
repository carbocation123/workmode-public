# 本地参考归档

该目录用于保存不能或不应提交到开源仓库的本地研究材料，例如真实论文、第三方皮肤解包、游戏截图、旧协议草稿和教程早期实验记录。

除本说明外，`local-reference/` 下的内容被 Git 忽略。正式源码、测试、构建和发行流程不得依赖这里的文件。需要把某项素材用于产品时，必须先确认来源与许可证，再迁入适当的可追踪目录并补齐文档。

当前分区：

- `skin-references/`：第三方视觉参考、解包素材和依赖这些素材的本地样稿；
- `skin-drafts/`：已退役或尚未迁移到当前协议的皮肤草稿；
- `tutorial-history/`：真实文献、合成文献、教程分析结果和早期模板记录。

这里是本地档案，不是缓存。日常清理不得批量删除；若确认某项已完成许可证审查并形成正式替代物，应先迁移或另行归档，再处理旧副本。
