# tools/

## 用途

保存可以跨项目复用、具有独立配置或外部服务边界的科研工具。项目专用、只处理本项目数据的脚本放在 `scripts/`。

## 当前工具

### `objective-facts/`

真实 PDF 客观事实提取工具链：

- MinerU 版面解析；
- 印刷页码映射与模型输入预处理；
- OpenAI Chat Completions 兼容模型的六段式报告生成；
- 一键串联脚本、配置模板和无网络测试。

入口与安全说明见 `objective-facts/README.md`。

### `epr/`

模拟 Bruker EPR 文件工具链：

- 生成可复现、明确标记为模拟的 PAR/SPC 教学样本；
- 把 little-endian float32 SPC 转成两列 ASC；
- 用逐谱微波频率换算 g 轴，报告噪声门限和客观谱形；
- 不做自动物种归属。

入口、预生成结果和科研边界见 `epr/README.md`。

## 纪律

- 工具目录必须有 README，说明输入、输出、依赖、配置、外部传输和验证方式。
- 配置模板不得包含真实 key；本地密钥文件必须被忽略。
- 工具产生的结果进入对应业务目录，本目录不堆放论文、报告或临时输出。
- 修改工具后同步更新本索引、项目根 README 和相关业务目录说明。
