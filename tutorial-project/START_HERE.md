# 从这里开始

这是一个可以安全修改的教学项目。样品和仪器数据均为虚构或合成内容；`papers/s41467-019-13638-9.pdf` 是一篇依据 CC BY 4.0 分发的真实开放获取论文，用作客观事实提取练习材料。

## 第一次打开

如果还没有模型 API，先打开「设置 → 模型 API → 如何申请 DeepSeek API」：

1. 登录 DeepSeek 开放平台；
2. 充值 API 余额；
3. 创建 API Key，并只粘贴到 Workmode Public 的密码输入框；
4. 点击「一键填入 V4 Pro」和「测试连接」。

不要把 API Key 写进本项目、聊天正文、截图或共享文档。申请入口和当前模型配置以软件中链接的 DeepSeek 官方页面为准。

1. 在 Workmode Public 中把本文件夹注册为项目。
2. 确认项目根目录存在 `WORKMODE.md`。软件会把它作为本项目专属提示词自动加载，无需修改全局提示词或工作记忆。
3. 新建一个名为「科研协作教程」的会话。
4. 发送：

> 开始教程。

## 教学目标

你将用同一个虚构研究项目体验：

1. 文章润色；
2. 文献调研与 PDF 客观事实提取；
3. 实验规划；
4. Python 数据分析、模拟 EPR 二进制转换与科研文件联动维护。

完成后，AI 会说明为什么不建议把带有示例资料和教学会话的项目直接当作正式科研项目。

完成六项教程任务还会解锁装饰性皮肤「Origin Ring」。论文纸、实验室、深夜观测站、高对比和跟随系统无需解锁，可随时在「设置 → 外观与皮肤」使用。

以后回复“继续”时，也只推进一个演示单元。如果你确实不想学习过程，可以明确说“跳过教学，直接完成任务”。

需要重新体验时，使用项目栏中的「重置教程」。软件会先备份当前教程文件和项目状态，再恢复初始模板并新建空白会话。
