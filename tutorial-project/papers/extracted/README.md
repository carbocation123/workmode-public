# extracted/

每篇 PDF 使用独立子目录：

```text
<paper-stem>/
  full.md                              版面解析正文
  *_content_list.json                  带页面与内容块信息的结构化结果
  *_prepared.txt                       带印刷页码的模型输入
  *_客观事实抽取报告_v2.md             六段式客观事实报告
  batch_manifest.json                  模型批处理记录
```

解析正文和模型输入都是中间材料，客观事实报告才是供后续人工判断的整理结果。所有输出都必须反向注明来源 PDF；报告中的事实必须带页码、Figure 或 Table 定位，第 6 节留给主对话综合，不由批处理模型代填。

教学预计算结果必须同时标明：是否真实调用外部服务、是否覆盖全文、使用了哪个提示词和由谁复核。模拟快照不得使用真实 API 的措辞冒充本轮运行结果。
